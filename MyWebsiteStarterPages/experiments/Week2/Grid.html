Grid layout and responsive design

  • Displays grid design where you can divide screen into different columns using div and class properties.
  • Grid system is created using stylesheet with width property.
  • As you decrease the size of tthe screen width of the grid layout changes
  • I have targeted screens with 800px,600px and 400 px
  • Layout of the links also changed with respect to size of the screen
  • This experiment also have fixed header
