On click function handled through jquery

click me! to change color

$("h1").click(function (event) {
console.log("clicked")
$("h1").css("color", "red");
});

Description: "click me! to change color" triggers jquery to write to console and change the color of the text

Add and clear content from the page on click

Description: After click on "Add content" button new h3 tags are added one below other which is also an editable content.

"Clear" button clears all the element added to the page."Save" option saves th content created by the add content.

"Load" button loads the object representation of the content in console variable
