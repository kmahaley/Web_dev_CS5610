TITLE

Online Airlines Status Project-SkyNet

AUTHOR

Kartik Mahaley

Master's student of computer Science at Northeastern University, Boston, USA

kmahaley@ccs.neu.edu

ABSTRACT

This project is part of my academic course CS5610 under Prof.Rasala. It aims to provide all sort of information related to flights and airports. One can check flight performance and book ticket based on it. My site provides links to sites which provides cheaper deals related to holidays, airline ticketing. You can also check the airport details and its current weather condition. Googlemap API's are used to show live weather situation along with the geolocation.

INTRODUCTION AND OBJECTIVE

To prepare a website which will have all what I have learnt in class and in doing experiments. I have taken this airline project beacause it is the most real time data and crucial mode of transport in today's life.

FEATURES

Airports

All the codes, name and details about the airport

Flights

Delay index and flight ratings given by travelors

Weather

Weather conditons like visibility, wind temperature etc.

Google Maps

Geo location of the airports or flights

FUTURE PLANS

Real time location of the flights, alerts before arrival of the flight, populating all IATA code of active airports instead of typing in textbox

REQUIREMENTS

All you need to access this website full functionality, is IATA code for airports or airlines

DESIGN

IMPLEMENTATION

I have used below mentioned technologies to develope this website

  • HTML5
  • CSS3
  • Javascript
  • JQuery
  • Ajax

I have also learnt and used technologies like Angular JS, Angular Routing and Bootstrap in my experiments

Home Page



Home page have short summary about my feature of the website. This will also guide you get IATA code of airlines or airports.

I have given links to top airlines in world who have won the award, so that guest can directly book flight tickets. If they are looking for some package deals then there are couple of links to travel website too.

In other section I have given links to popular beach, holiday destinations

Airport Page



This page gives details about the airport and map location of it. User will enter IATA code to get details of the airport such as region, country, name.

After initial search there are option of details or map. Map will give the collapsable view of the airport where guest can zoom in or out, caan move the map. He can also check the weather of the airport which is provided by google APIs

There are two more options to get details of the airport, first will give you time related information with GMT while second will give you all the codes you will need for example postal code, city code, country code

Flights Page



On this page you can check the ratings of the flight given by the travellers, for example you want to check delay index of the flight or cancellation percentage.

This will also give you flight path as well as straight path. As google is yet to provide flight mode in googlemap this will simple give to source to destination polylines

It will list out departure and arrival airport details to you.

Weather Page



Search based on IATA code of any airport name. Each button triggers action on page. These tables or pane are collapsable with every button click due to toggle effect on them.

Google map API used here takes parameters from flightstats API to generate map. I have provided width and height for div which shows map


  • Brief details about the airport's country, city and name
  • My Map button gives the map with weather details using googlemap API
  • Weather button generate more data about the airport's weather condition such as temperature, presure, due point etc
  • Finally more button, will generate additional data about skyconditions such as sloudy or clear, wind conditions etc

Get IATA Page

This site provide you with details of the airport and it's code if you just remember your city of arrival or departure. Select name of the city and you will get IATA code.

Navigation bar



Navigation bar is modified such that similar features can be grouped together. Background and on hover color are kept different to enhance user navigation experience.

Review Flight and airport experience



Traveller can leave there feedback onto the page for airport they visited or flights in which they travelled. These comments are not stored in database so with each refresh they will erase out.

Stylesheet



Stylesheet are modified choosing contrast colo combinations. Sometimes APIs take time to load data, hence to show user that data is getting loaded onto page a message is displayed and it disappears on data load.

Validations



If user tries to enter blank data then error is thrown in form of alert saying input cannot be empty and input fields are marked in red.

Check airlines and flights between airports



After finalizing holiday destination we need a better way to travel there afterall its a vacation time. This feature enables travellers to see what airlines and what flights are offered between their holiday destinations They can always review the flight details using flight rating module.
